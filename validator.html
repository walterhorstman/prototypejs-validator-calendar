Initialization

    var validator;
    var calendar;
    document.observe('dom:loaded', function() {
      validator = new Validator();
      calendar = new Calendar();
    });

The constructor can have a Hash-like argument, with 4 special keys:

  • validators: extra/altered validators
  • country: 2 character country code, sets some country specific default options
  • language: 2 character language code, sets some language specific default options
  • options: altered default options

Enabling

To enable validation of fields in a form, you should add the class validateable to the form. The name of the class can be overwritten via option triggerClass (if this option is null, all forms will be validated).

Default validators

The next section is a description and test of the default validators.

Required fields

<input id="required" type="text" name="required" class="required" />

Confirmation of fields

<input id="confirmation" type="text" name="confirmation" class="confirmation:required" />

Date fields

<input id="date" type="text" name="date" class="date" />

Length of fields

<input id="minMaxLength" type="text" name="minMaxLength" class="minLength:5 maxLength:10" />

Numeric fields

<input id="numeric" type="text" name="numeric" class="numeric" />

Minimum and maximum values of fields

<input id="minMaxValue" type="text" name="minMaxValues" class="minValue:5 maxValue:10" />
